# dma-cli.sh

CLI to use the hidden service http server 
  
*parseCLI is a library to create commands/cli software using ts, js or bash available at [Github Simple Arguments Library](https://github.com/stringmanolo/simpleArgumentsParser) * 

### Hidden Service
This hidden service is hardcoded, do not trust this address since its available to the public. Only for testing. Do not trust its output, generate your own for testing using this repo [Dark Messenger for Terminal](https://github.com/StringManolo/darkmessenger)
